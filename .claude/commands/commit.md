Prepare for commit.

## Step 1: Format
```bash
npm run format
```

## Step 2: Lint
```bash
npm run lint
```

## Step 3: Test
```bash
npm test
```

## Step 4: Review
Quick check:
- Debug code removed?
- Ready to commit?

## Step 5: Message
Format:
```
[verb] [what changed]

- Detail
- Detail

Fixes #123
```

Verbs: Add, Fix, Update, Remove, Refactor
