## Step 1: Design
Use the **ui-designer** agent to:
- Define component structure
- Plan props/API
- Consider all states:
  - Default
  - Loading
  - Error
  - Empty
  - Disabled

## Step 2: Accessibility
Use the **accessibility-checker** agent to plan:
- Keyboard navigation
- ARIA attributes
- Focus management
- Screen reader support

## Step 3: Style
Use the **css-wizard** agent for:
- Component styling
- Responsive behavior
- Theme support
- Animations if needed

## Step 4: Implement
Build the component:
- Start with structure
- Add styling
- Add interactivity
- Handle all states

## Step 5: Test
Use the **test-writer** agent for:
- Render tests
- Interaction tests
- Accessibility tests
- Edge case tests

## Step 6: Document
Use the **docs-writer** agent for:
- Props documentation
- Usage examples
- Storybook story (if applicable)
