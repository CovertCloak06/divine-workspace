Generate changelog.

## Step 1: Gather
Collect changes since last release:
- Git commits
- Merged PRs
- Closed issues

## Step 2: Categorize
Use the **changelog-writer** agent:

### Added
New features

### Changed
Changes to existing features

### Fixed
Bug fixes

### Removed
Removed features

### Security
Security fixes

### Breaking
Changes requiring user action

## Step 3: Format
Follow Keep a Changelog format.
Focus on user impact, not implementation.
